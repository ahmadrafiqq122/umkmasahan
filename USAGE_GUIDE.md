# Panduan Penggunaan - Peta Digital UMKM Asahan

## Untuk Pengunjung (Publik)

### Melihat Peta Digital
1. Buka website di browser
2. Halaman utama akan menampilkan peta Kabupaten Asahan
3. Marker biru menunjukkan lokasi usaha yang telah disetujui

### Melihat Detail Usaha
1. Klik marker pada peta
2. Pop-up akan muncul dengan informasi singkat
3. Klik tombol "Lihat Detail" untuk informasi lengkap
4. Detail meliputi: nama usaha, pemilik, kontak, alamat, foto, dll

### Mencari Usaha
1. Gunakan form pencarian di atas peta
2. Masukkan kata kunci, pilih jenis usaha, atau pilih kecamatan
3. Klik tombol "Cari"
4. Peta akan update menampilkan hasil pencarian

## Untuk Pelaku Usaha

### Registrasi Akun Baru

1. **Klik "Daftar" di menu navigasi**

2. **Isi form registrasi:**
   - Nama lengkap
   - Email (akan digunakan untuk verifikasi)
   - Nomor telepon
   - NIK (opsional)
   - Alamat (opsional)
   - Password (minimal 8 karakter)
   - Konfirmasi password

3. **Klik "Daftar Sekarang"**

4. **Verifikasi Email:**
   - Cek email Anda (termasuk folder spam)
   - Salin kode verifikasi 6 digit
   - Masukkan kode di halaman verifikasi
   - Klik "Verifikasi Sekarang"

5. **Jika tidak menerima email:**
   - Klik "Kirim Ulang Kode"
   - Tunggu beberapa menit

### Login

1. Klik "Login" di menu navigasi
2. Masukkan email dan password
3. Centang "Ingat Saya" jika ingin tetap login
4. Klik tombol "Login"

### Menambah Data Usaha

1. **Login ke akun Anda**

2. **Klik "Tambah Data Usaha Baru" di dashboard**

3. **Isi Informasi Dasar:**
   - Nama usaha
   - Nama pemilik
   - Jenis usaha (pilih dari dropdown)
   - Skala usaha (Mikro/Kecil/Menengah)
   - Tahun berdiri
   - Deskripsi usaha (jelaskan dengan detail)
   - Produk utama
   - Jumlah karyawan
   - Omzet per bulan (opsional)

4. **Isi Informasi Kontak:**
   - Nomor telepon (wajib)
   - WhatsApp
   - Email usaha
   - Instagram (contoh: @namausaha)
   - Facebook
   - Website

5. **Isi Alamat & Lokasi:**
   - Alamat lengkap
   - Desa/Kelurahan
   - Kecamatan (pilih dari dropdown)
   - Kode pos
   - **Klik pada peta untuk menandai lokasi usaha**
   - Koordinat akan terisi otomatis

6. **Isi Legalitas (jika ada):**
   - NIB (Nomor Induk Berusaha)
   - SIUP

7. **Upload Foto:**
   - Klik "Choose File" untuk upload foto
   - Pilih tipe foto (Tempat Usaha/Produk/Lainnya)
   - Tambah keterangan jika perlu
   - Klik "Tambah Foto" untuk upload lebih banyak
   - **Minimal 1 foto wajib diupload**

8. **Klik "Simpan Data Usaha"**

9. **Status awal: PENDING (Menunggu Persetujuan Admin)**

### Melihat Status Usaha

1. Login ke dashboard
2. Lihat statistik di atas:
   - **Total**: Jumlah usaha yang Anda daftarkan
   - **Disetujui**: Usaha yang sudah tampil di peta
   - **Menunggu**: Usaha dalam proses verifikasi
   - **Ditolak**: Usaha yang perlu diperbaiki

3. Daftar usaha menampilkan:
   - Nama usaha dan pemilik
   - Jenis dan kecamatan
   - Jumlah foto
   - Status (badge berwarna)
   - Tanggal pendaftaran
   - Tombol aksi

### Mengedit Data Usaha

1. Di dashboard, klik tombol **Edit** (ikon pensil)
2. Update informasi yang perlu diubah
3. Update foto jika perlu (hapus/tambah)
4. Klik "Update Data Usaha"
5. **Jika status sebelumnya "Ditolak", status akan kembali "Pending"**

### Menghapus Data Usaha

1. Di dashboard, klik tombol **Hapus** (ikon sampah)
2. Konfirmasi penghapusan
3. Data akan terhapus permanen

### Mengelola Data Usaha

#### Melihat Semua Usaha
1. Menu: **Kelola Data Usaha**
2. Gunakan filter:
   - Cari by keyword
   - Filter by status
   - Filter by jenis usaha
   - Filter by kecamatan
3. Klik "Filter"

1. Klik tombol **Edit** (pensil kuning)
2. Update informasi yang perlu diubah
3. Klik "Update Data Usaha"

#### Hapus Data Usaha
1. Klik tombol **Hapus** (sampah merah)
2. Konfirmasi penghapusan

#### Bulk Operations
1. **Centang** checkbox usaha yang ingin diproses
2. Untuk menyetujui banyak usaha: Klik "Setujui Terpilih"
3. Untuk menghapus banyak usaha: Klik "Hapus Terpilih"

### Mengelola User

#### Melihat Semua User
1. Menu: **Kelola User**
2. Gunakan filter:
   - Cari by keyword
   - Filter by role (Admin/User)
   - Filter by status (Aktif/Nonaktif)

#### Melihat Detail User
1. Klik tombol **Detail** (mata biru)
2. Lihat informasi user dan daftar usahanya

#### Tambah User Baru
1. Klik tombol **"Tambah User"**
2. Isi form:
   - Nama, email, password
   - Role (Admin atau User)
   - Nomor telepon
   - NIK dan alamat (opsional)
3. Klik "Simpan"
4. **User langsung aktif tanpa verifikasi email**

#### Edit User
1. Klik tombol **Edit** (pensil kuning)
2. Update informasi
3. Kosongkan password jika tidak ingin mengubah
4. Klik "Simpan"

#### Aktifkan/Nonaktifkan User
1. Klik tombol **‚è∏** (pause) untuk nonaktifkan
2. Klik tombol **‚ñ∂** (play) untuk aktifkan
3. User nonaktif tidak dapat login

#### Hapus User
1. Klik tombol **Hapus** (sampah merah)
2. **Tidak bisa hapus jika user punya data usaha**
3. **Tidak bisa hapus akun sendiri**

### Best Practices untuk Admin

1. **Review dengan Teliti**
   - Cek kesesuaian foto dengan deskripsi
   - Verifikasi lokasi di peta
   - Pastikan kontak valid

2. **Berikan Alasan Jelas saat Menolak**
   - Jelaskan bagian mana yang bermasalah
   - Berikan saran perbaikan
   - Gunakan bahasa yang sopan

3. **Responsif**
   - Cek pending approval secara rutin
   - Proses persetujuan maksimal 1-3 hari

4. **Backup Data**
   - Lakukan backup database secara berkala
   - Simpan di tempat yang aman

## Tips Umum

### Keamanan Akun
- Gunakan password yang kuat (kombinasi huruf besar, kecil, angka)
- Jangan bagikan password ke orang lain
- Logout setelah selesai menggunakan

### Browser yang Disarankan
- Google Chrome (latest)
- Mozilla Firefox (latest)
- Microsoft Edge (latest)
- Safari (latest)

### Troubleshooting

**Lupa Password?**
- Hubungi admin via email: disperindagkop@asahankab.go.id

**Email Verifikasi Tidak Masuk?**
- Cek folder Spam/Junk
- Tunggu 5-10 menit
- Klik "Kirim Ulang Kode"
- Pastikan email yang dimasukkan benar

**Foto Tidak Bisa Diupload?**
- Cek ukuran file (maksimal 2MB)
- Format harus JPG, PNG, atau JPEG
- Gunakan foto dengan resolusi wajar

**Koordinat Tidak Terisi?**
- Pastikan Anda sudah **klik pada peta**
- Browser mungkin memblok lokasi, refresh page

**Peta Tidak Muncul?**
- Cek koneksi internet
- Refresh halaman (F5)
- Clear cache browser
